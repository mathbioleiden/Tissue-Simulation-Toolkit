Detailed Description

Add new adhesion particles

These particles may represent various aspects of the cell, adhesions, etc. depending on the algorithm. Technically however, they are marked as type "adhesion", meaning that their position is controlled by the CPM and that the ECM will take them into account, but not move them.

After the particles are added, for any particles that have a value >0.0 for bond_attempt_radius an attempt will be made to find a "free" type particle within that radius. If one is found, a new bond will be made between them. This represents the cell actually adhering to the ECM.
