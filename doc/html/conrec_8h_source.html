55 // CONREC is a contouring subroutine for rectangularily spaced data.
56 //
57 // It emits calls to a line drawing subroutine supplied by the user
58 // which draws a contour map corresponding to real*4data on a randomly
59 // spaced rectangular grid. The coordinates emitted are in the same
60 // units given in the x() and y() arrays.
61 //
62 // Any number of contour levels may be specified but they must be
63 // in order of increasing value.
64 //
65 // As this code is ported from FORTRAN-77, please be very careful of the
66 // various indices like ilb,iub,jlb and jub, remeber that C/C++ indices
67 // starts from zero (0)
68 //
69 //=============================================================================
